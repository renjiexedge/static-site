# Static Site

This project is a simple static website that includes an HTML document, CSS styles, and JavaScript functionality.

## Project Structure

```
static-site
├── index.html       # Main HTML document
├── script.js        # JavaScript file for interactivity
└── style.css        # CSS file for styling
```

## Getting Started

To view the static site, open the `index.html` file in a web browser. 

## Features

- Basic HTML structure
- Responsive design using CSS
- Interactive elements powered by JavaScript

## Installation

No installation is required. Simply clone the repository and open `index.html` in your preferred web browser.
